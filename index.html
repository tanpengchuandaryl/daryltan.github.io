Featured Project

Limo ROS Navigation
May 2025 - Jul 2025
Limo Robot ROS Python SLAM RTAB-Map AMCL EKF tf2

This project demonstrates advanced autonomous navigation capabilities using a Limo Robot platform integrated with ROS (Robot Operating System). The system implements sophisticated SLAM (Simultaneous Localization and Mapping) techniques to enable real-time environment mapping and precise robot localization.

Key technical implementations include RTAB-Map for robust 3D mapping, AMCL (Adaptive Monte Carlo Localization) for probabilistic position estimation, and Extended Kalman Filter (EKF) for sensor fusion and state estimation. The tf2 transformation library ensures accurate coordinate frame management throughout the navigation stack.

The project showcases practical applications of robotics in real-world scenarios, combining theoretical knowledge with hands-on implementation of industry-standard navigation algorithms.
